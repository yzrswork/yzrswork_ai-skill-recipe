SE3 既知バグ対応

iOS 26で(i)ボタンから着信拒否できない問題

iPhone SE3ではiOS 26以降、電話アプリの履歴から「連絡先を着信拒否」が表示されないバグが報告されています。 以下の方法で回避できます。

方法A: 設定画面を直接開く Direct Link

下のボタンで「着信拒否した連絡先」の設定画面を直接開きます。 開いたら右上の「+」をタップして番号を追加。合計2タップ。

着信拒否リストを開く (A) App-prefs:Privacy&path=BLOCKED -> プライバシーとセキュリティを開く (B) prefs:root=Privacy -> 電話 > 着信拒否を開く (C) App-prefs:Phone&path=BLOCKED ->
使い方: 上の3つのリンクを順にタップして、設定アプリが開くか試してください。 開いたリンクをSafariのブックマークに追加し、ホーム画面に置けばワンタップで使えます。

iOS版で異なる場合: URLスキームはiOSのマイナーバージョンで変わることがあります。 どれも開かない場合は、下の「方法B」を使ってください。
方法B: ショートカットで1タップ化 Shortcut
ショートカットの作り方
  1. ショートカットアプリを開き、右上の「+」で新規作成
  2. 「アクションを追加」をタップ → 検索で「URLを開く」を選択
  3. URLに上のリンクで動いたURLを入力
    App-prefs:Privacy&path=BLOCKED
  4. 上部のショートカット名を「着信拒否」に変更
  5. 名前の横の「V」をタップ →
    「ホーム画面に追加」を選択
完成後: ホーム画面のアイコンを1タップ → 設定の着信拒否リストが開く → 「+」で番号追加。 電話アプリの(i)ボタンを経由しないので、SE3のバグに影響されません。
方法C: Classic表示で(i)ボタン復活 Workaround

電話アプリの表示を「クラシック」に切り替えると(i)ボタンが復活する場合があります。 ただしSE3では追加のステップが必要です。

  1. 電話アプリの履歴タブを開く
  2. 右上の 三本線アイコン をタップ →「クラシック」を選択
  3. ブロックしたい番号の(i)をタップ
  4. 画面が空白のまま → ホームボタンを押してホーム画面に戻る
  5. 電話アプリに戻ると、項目が表示される。 一番下の「連絡先を着信拒否」をタップ
注意: この方法はSE3特有のワークアラウンドです。 (i)をタップ後に空白になるのはバグで、ホームボタンで一旦離れて戻ると表示が修復されます。 手順が多いので、方法A/Bの方が快適です。
最も効果的

iOS 26「通話スクリーニング」を有効にする

一度だけ設定すれば、連絡先にない番号からの着信を自動でスクリーニング。 番号ごとにブロックする手間がなくなります。

設定手順 Setup
  1. 設定アプリを開く
  2. アプリ をタップ
  3. 電話 をタップ
  4. 下へスクロールし
    不明な発信者をスクリーニング をタップ
  5. モードを選択（下記参照）
3つのモード Modes
おすすめ
通話の理由を尋ねる
不明な番号から着信すると、iPhoneが自動応答して相手に名前と用件を聞きます。 内容がテキストで画面に表示され、出るかどうかを判断できます。 正当な電話（配達、病院等）はちゃんと繋がるので安心。
消音
不明な番号からの着信を音なしで留守番電話へ直送。 履歴には残るので後から確認可能。最も強力だが、 初めての配達業者等も無音になる点に注意。
なし（オフ）
スクリーニングを無効化。全ての着信が通常どおり鳴ります。
補足: スクリーニングは連絡先に登録済みの番号には適用されません。 家族、友人、取引先は事前に連絡先に登録しておけば通常どおり着信します。

通話の理由を尋ねるモード: 迷惑電話業者はほぼ100%ここで切れます。 正当な用件のある相手だけが録音を残すので、事実上の自動選別になります。
