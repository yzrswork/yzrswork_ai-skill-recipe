迷惑電話
ブロッカー-- Spam Call Blocker

SE3で消えた着信拒否ボタンの代わり。番号を入れて連絡先を即作成、そこからブロック。

SE3 既知バグ対応

iOS 26で(i)ボタン/左スワイプから着信拒否できない問題

SE3ではPhone appの履歴画面のブロック機能にバグがありますが、 連絡先アプリからのブロックは正常に動作します。 番号を連絡先として即作成し、そこからブロックする方式で回避します。

即ブロック Quick Block

番号を入力して「連絡先を作成」をタップ。ダウンロードされたファイルを開くと 連絡先アプリが起動するので、保存してから「この発信者を着信拒否」で完了。

手順 Flow
  1. 電話アプリの履歴で迷惑番号を長押し → コピー
  2. このページに戻り「クリップボードから貼付」→「連絡先を作成」
  3. ダウンロードされた.vcfファイルを開く（Safari下部のダウンロードボタン、または通知から）
  4. 連絡先アプリが開くので「新規連絡先を作成」→「完了」で保存
  5. 保存した連絡先を開き、一番下の「この発信者を着信拒否」をタップ
ポイント: Phone appの履歴画面のブロック機能にバグがあるだけで、 連絡先アプリの「この発信者を着信拒否」は正常に動作します。 連絡先を経由することでバグを完全に回避できます。
更に時短: ショートカット自動化 Shortcut

クリップボードの番号から連絡先を自動作成するショートカットを作れば、 このページを開く手間も省けます。

ショートカットの作り方
  1. ショートカットアプリ → 右上「+」で新規作成
  2. 「アクションを追加」 → 検索で「クリップボードを取得」を追加
  3. 続けて「新規連絡先を追加」アクションを追加。
    電話番号に「クリップボード」変数を設定、
    姓に「SPAM」と入力
  4. ショートカット名を「迷惑電話ブロック」に変更 →
    「V」→「ホーム画面に追加」
使い方: 履歴で番号をコピー → ホーム画面のアイコンをタップ → 連絡先が自動作成される → 連絡先アプリで開いて「着信拒否」。
別の方法: Classic表示ワークアラウンド Fallback
  1. 電話アプリ → 履歴タブ
  2. 右上の 三本線アイコン →「クラシック」を選択
  3. ブロックしたい番号の(i)をタップ
  4. 画面が空白のまま → ホームボタンを押してホーム画面に戻る
  5. 電話アプリに戻ると項目が表示される →
    一番下の「連絡先を着信拒否」をタップ
SE3固有: (i)タップ後の空白はバグです。 ホームボタンで離れて戻ると表示が修復されます。
最も効果的

iOS 26「通話スクリーニング」を有効にする

一度だけ設定すれば、連絡先にない番号からの着信を自動でスクリーニング。 番号ごとにブロックする手間がなくなります。

設定手順 Setup
  1. 設定アプリを開く
  2. アプリ をタップ
  3. 電話 をタップ
  4. 下へスクロールし
    不明な発信者をスクリーニング をタップ
  5. モードを選択（下記参照）
3つのモード Modes
おすすめ
通話の理由を尋ねる
不明な番号から着信すると、iPhoneが自動応答して相手に名前と用件を聞きます。 内容がテキストで画面に表示され、出るかどうかを判断できます。 正当な電話（配達、病院等）はちゃんと繋がるので安心。
消音
不明な番号からの着信を音なしで留守番電話へ直送。 履歴には残るので後から確認可能。最も強力だが、 初めての配達業者等も無音になる点に注意。
なし（オフ）
スクリーニングを無効化。全ての着信が通常どおり鳴ります。
補足: スクリーニングは連絡先に登録済みの番号には適用されません。 家族、友人、取引先は事前に連絡先に登録しておけば通常どおり着信します。

通話の理由を尋ねるモード: 迷惑電話業者はほぼ100%ここで切れます。 正当な用件のある相手だけが録音を残すので、事実上の自動選別になります。
