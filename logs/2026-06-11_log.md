## やったこと

### 1. ウィルスバスターズ（`virus-busters.html`）の新規作成
- 外部依存なしの単一HTML。Canvas描画（内部270×480 / pixelated拡大）+ Web Audio。
- iPhoneのSafari縦持ちで片手プレイできる前進ランナー型ディフェンスゲーム。

### 2. ゲーム仕様
- 画面を左右ドラッグで白血球の群れを操作。射撃・前進は自動。
- 全20ステージ（鼻腔→のど→気管→肺→血管→…と体内を進むテーマ）。
- 難易度4種: easy / normal / hard / WTF（WTFはHP×3.4・物量×2.6の鬼畜仕様）。
- 仲間上限2000人。上限後は売店の永続強化で火力を伸ばす設計。

### 3. ギミック
- `+N`/`×N`の半透明青ゲート（撃つと+値が増える・通過で仲間増員）
- コンテナ破壊でアイテム（武器強化・コイン・仲間増員）ドロップ
- 武器強化7段階（単発→多WAY→直線ビーム→ビーム幅拡大）、段階ごとに弾の色・エフェクト変化。強化は群れ全員に適用。

### 4. 敵・ボス
- ウィルス図鑑51種: 雑魚は実在ウィルス名ベース（インフル/コロナ/ノロ/はしか等）＋色違い強化版。
- ボス10種は病名（鼻炎こぞう・気管支炎男爵・髄膜炎魔王など）でドラクエのボストロール風。
- 中ボス・ラスボスともにHPバー・弾幕・子分召喚あり。

### 5. UI・サウンド
- テラコッタ配色（`index.html` と共通の `--terracotta` 系）。
- Web AudioでチップチューンBGM（タイトル/道中/ボス）＋効果音10種。ミュート可。
- localStorageに自動保存（クリア状況・コイン・図鑑・強化）、リセットボタンあり。

### 6. `index.html` / `README.md`
- 作品一覧に「09 ウィルスバスターズ — Virus Busters」カードを追加。

## 追記セッション（2026-06-11 後半）

### Metronome Watch（`metronome-watch.html`）の新規作成・PR #26 マージ済み

- ブランチ: `claude/metronome-watch-app-9wcn1q`

#### アプリ仕様

**時計文字盤**
- 外部依存なしの単一HTML（7KB弱、オフライン動作）。
- 64目盛（16小節 × 4拍）の円形文字盤を SVG で描画。
- 小節の1拍目は長く太い明るいシアン目盛で強調、内側に小節番号 1〜16（1/5/9/13 は大きく発光）。
- 12時位置に白いダイヤマーカー（小節1の頭）。
- 針はクォーツ時計風に1拍ごとステップ移動（1周 = 16小節 = 64拍）、現在目盛は白く点灯。
- 中央に BPM 大表示、`BAR 03 / 16 · BEAT 2` テキスト、4拍ドット（1拍目白フラッシュ）、リアルタイム時刻（ウォッチ表示）。

**メトロノーム機能**
- BPM 30〜300：スライダー / ±ボタン（長押しで連続加減） / TAP テンポ（直近5タップ平均）。
- クリック音 5種（BEEP / CLICK / WOOD / RIM / BELL）— Web Audio API で合成、チップタップで試聴。
- カウントイン OFF / 1小節 / 2小節（専用音＋中央 4→3→2→1 大表示）。
- 1拍目アクセント ON/OFF、音量スライダー。
- 設定は localStorage に保存し次回起動時に復元。

**精度**
- setInterval ではなくルックアヘッド方式（25ms タイマー × 120ms 先スケジュール）。
- AudioContext タイムラインにスケジュールするため描画負荷に左右されない。

**演出**
- 時計背面の透明な大きな振り子：再生中は32拍（= 8小節）で一往復（拍位相に同期）、停止中は自走。
- 背景：流れる星屑 + パースグリッドのサイバー空間（Canvas）、拍ごとにリングフラッシュ。
- 装飾リング2本がゆっくり逆回転するアニメーション。

**iPhone 対応**
- 初回タッチで AudioContext 解錠。
- 再生中に Screen Wake Lock を取得（バックグラウンド復帰時に再取得）。
- safe-area / viewport-fit=cover / apple-mobile-web-app-capable 設定済み。
