## やったこと

### 1. ウィルスバスターズ（`virus-busters.html`）の新規作成
- 外部依存なしの単一HTML。Canvas描画（内部270×480 / pixelated拡大）+ Web Audio。
- iPhoneのSafari縦持ちで片手プレイできる前進ランナー型ディフェンスゲーム。

### 2. ゲーム仕様
- 画面を左右ドラッグで白血球の群れを操作。射撃・前進は自動。
- 全20ステージ（鼻腔→のど→気管→肺→血管→…と体内を進むテーマ）。
- 難易度4種: easy / normal / hard / WTF（WTFはHP×3.4・物量×2.6の鬼畜仕様）。
- 仲間上限2000人。上限後は売店の永続強化で火力を伸ばす設計。

### 3. ギミック
- `+N`/`×N`の半透明青ゲート（撃つと+値が増える・通過で仲間増員）
- コンテナ破壊でアイテム（武器強化・コイン・仲間増員）ドロップ
- 武器強化7段階（単発→多WAY→直線ビーム→ビーム幅拡大）、段階ごとに弾の色・エフェクト変化。強化は群れ全員に適用。

### 4. 敵・ボス
- ウィルス図鑑51種: 雑魚は実在ウィルス名ベース（インフル/コロナ/ノロ/はしか等）＋色違い強化版。
- ボス10種は病名（鼻炎こぞう・気管支炎男爵・髄膜炎魔王など）でドラクエのボストロール風。
- 中ボス・ラスボスともにHPバー・弾幕・子分召喚あり。

### 5. UI・サウンド
- テラコッタ配色（`index.html` と共通の `--terracotta` 系）。
- Web AudioでチップチューンBGM（タイトル/道中/ボス）＋効果音10種。ミュート可。
- localStorageに自動保存（クリア状況・コイン・図鑑・強化）、リセットボタンあり。

### 6. `index.html` / `README.md`
- 作品一覧に「09 ウィルスバスターズ — Virus Busters」カードを追加。
