### 1. ウィルスバスターズ（`virus-busters.html`）の新規作成
- 外部依存なしの単一HTML。Canvas描画（内部270×480 / pixelated拡大）+ Web Audio。
- iPhoneのSafari縦持ちで片手プレイできる前進ランナー型ディフェンスゲーム。

### 2. ゲーム仕様
- 画面を左右ドラッグで白血球の群れを操作。射撃・前進は自動。
- 全20ステージ（鼻腔→のど→気管→肺→血管→…と体内を進むテーマ）。
- 難易度4種: easy / normal / hard / WTF（WTFはHP×3.4・物量×2.6の鬼畜仕様）。
- 仲間上限2000人。上限後は売店の永続強化で火力を伸ばす設計。

### 3. ギミック
- `+N`/`×N`の半透明青ゲート（撃つと+値が増える・通過で仲間増員）
- コンテナ破壊でアイテム（武器強化・コイン・仲間増員）ドロップ
- 武器強化7段階（単発→多WAY→直線ビーム→ビーム幅拡大）、段階ごとに弾の色・エフェクト変化。強化は群れ全員に適用。

### 4. 敵・ボス
- ウィルス図鑑51種: 雑魚は実在ウィルス名ベース（インフル/コロナ/ノロ/はしか等）＋色違い強化版。
- ボス10種は病名（鼻炎こぞう・気管支炎男爵・髄膜炎魔王など）でドラクエのボストロール風。
- 中ボス・ラスボスともにHPバー・弾幕・子分召喚あり。

### 5. UI・サウンド
- テラコッタ配色（`index.html` と共通の `--terracotta` 系）。
- Web AudioでチップチューンBGM（タイトル/道中/ボス）＋効果音10種。ミュート可。
- localStorageに自動保存（クリア状況・コイン・図鑑・強化）、リセットボタンあり。

### 6. `index.html` / `README.md`
- 作品一覧に「09 ウィルスバスターズ — Virus Busters」カードを追加。

## 追記セッション（2026-06-11 後半）

### Metronome Watch（`metronome-watch.html`）の新規作成・PR #26 マージ済み

- ブランチ: `claude/metronome-watch-app-9wcn1q`

#### アプリ仕様

**時計文字盤**
- 外部依存なしの単一HTML（7KB弱、オフライン動作）。
- 64目盛（16小節 × 4拍）の円形文字盤を SVG で描画。
- 小節の1拍目は長く太い明るいシアン目盛で強調、内側に小節番号 1〜16（1/5/9/13 は大きく発光）。
- 12時位置に白いダイヤマーカー（小節1の頭）。
- 針はクォーツ時計風に1拍ごとステップ移動（1周 = 16小節 = 64拍）、現在目盛は白く点灯。
- 中央に BPM 大表示、`BAR 03 / 16 · BEAT 2` テキスト、4拍ドット（1拍目白フラッシュ）、リアルタイム時刻（ウォッチ表示）。

**メトロノーム機能**
- BPM 30〜300：スライダー / ±ボタン（長押しで連続加減） / TAP テンポ（直近5タップ平均）。
- クリック音 5種（BEEP / CLICK / WOOD / RIM / BELL）— Web Audio API で合成、チップタップで試聴。
- カウントイン OFF / 1小節 / 2小節（専用音＋中央 4→3→2→1 大表示）。
- 1拍目アクセント ON/OFF、音量スライダー。
- 設定は localStorage に保存し次回起動時に復元。

**精度**
- setInterval ではなくルックアヘッド方式（25ms タイマー × 120ms 先スケジュール）。
- AudioContext タイムラインにスケジュールするため描画負荷に左右されない。

**演出**
- 時計背面の透明な大きな振り子：再生中は32拍（= 8小節）で一往復（拍位相に同期）、停止中は自走。
- 背景：流れる星屑 + パースグリッドのサイバー空間（Canvas）、拍ごとにリングフラッシュ。
- 装飾リング2本がゆっくり逆回転するアニメーション。

**iPhone 対応**
- 初回タッチで AudioContext 解錠。
- 再生中に Screen Wake Lock を取得（バックグラウンド復帰時に再取得）。
- safe-area / viewport-fit=cover / apple-mobile-web-app-capable 設定済み。

#### ナビゲーション修正（PRマージ後の追加コミット）
- マージ後、アプリ右上の「← INDEX」戻りリンクが9px薄グレー文字で視認困難だったため修正。
- ガラスパネル + ネオン枠の丸型チップボタン（タップ領域 34px 以上、シアン発光）に変更。

#### コミット

# セッション2: YZRsynth_v3.1（TR-808刷新）

## 概要
`tr808.html` を全面刷新して `yzrsynth.html` にリネーム。アプリ名を **YZRsynth_v3.1** に変更。  
TR-909実機風配色・16ステップドラムシーケンサー・スケールロック+グライド・AudioWorklet化・ビジュアライザ削除+3×2パッドグリッドなど複数機能を追加。

- ブランチ: `claude/tr-808-ui-redesign-6yiuut`
- PR: [#27](https://github.com/yzrswork/yzrswork_ai-skill-recipe/pull/27)

## やったこと

### 1. ダークミニマルUIへの刷新（フリーズ対策含む）
- テラコッタ配色を廃止し、TR-909実機風配色に変更
  - `--chassis: #161614`（黒シャーシ）
  - `--panel: #d7d4cc`（ライトグレー操作面）
  - `--accent: #f07818`（オレンジアクセント）
  - `--led: #e0301e`（赤LED）
- ゲームモード廃止
- フリーズ対策: ScriptProcessorNodeを録音時のみ接続、`visibilitychange` 時にノート全解放・シーケンサー停止、AudioContext自動resume

### 2. ファイルリネーム (`tr808.html` → `yzrsynth.html`)
- `git mv` で `tr808.html` → `yzrsynth.html`
- 旧URLへのリダイレクトスタブ（`tr808.html`）を作成: `location.replace` + `meta refresh` でハッシュごと転送
- `index.html` / `README.md` / `REFACTORING_PLAN.md` のリンク・説明文を全て更新

### 3. アプリ名変更: YZRsynth_v3.1
- `<title>` タグ、ヘッダー表示名を更新
- index.html の作品カードタイトル・説明文を更新

### 4. 16ステップドラムシーケンサー追加
- 5トラック構成: BD / SD / CH / OH / CP（タブ切替で1トラックずつ編集）
- 全合成音源（サンプル不使用）
  - BD: サイン波 150→50Hz ピッチエンベロープ
  - SD: バンドパスノイズ + 180Hzトーン
  - CH/OH: ハイパスノイズ + 矩形波倍音（減衰長の違いで2種）
  - CP: ノイズ多段バースト（15ms間隔×3）
- "A Tale of Two Clocks" パターンのlookahead scheduler（setInterval 25ms、0.1秒先予約）
- SWING ノブ（0–60%）でグルーヴ感調整
- ドラムバスは `masterGain` 直結（ルーパーで録音可能なシナジー設計）

### 5. スケールロック + グライド
- ヘッダーにセグメントコントロール: `FREE / PENTA / MAJ`
  - PENTA: Aマイナーペンタトニック `[57,60,62,64,67,69]`
  - MAJ: Cメジャー `[60,62,64,65,67,69]`
- スケール選択時は鍵盤を等幅パッドに再構築（黒鍵なし、音名ラベル付き）
- GLIDE トグル: スライド操作でリトリガーせず `setTargetAtTime` でピッチ滑走

### 6. AudioWorklet化（フリーズ対策の本筋）
- インラインコード文字列 → Blob URL → `audioWorklet.addModule()` で単一ファイル維持
- オーディオスレッドでリングバッファ処理、録音完了時に `Float32Array` を transferable で返送
- Safari 14.0以前の非対応環境は ScriptProcessorNode フォールバック

### 7. ノブ操作性改善
- ドラッグ中に値バブル表示（「あとどれだけ動かすか」の視覚フィードバック）
- ダブルタップでデフォルト値にリセット

### 8. URL設定共有機能
- `location.hash` にknob値・音色・スケール・GLIDE・BPM/スイング・ドラムパターン（hex）を直列化
- 変更時に `history.replaceState` で反映、読み込み時に復元

### 9. ルーパー強化
- CLR ボタン二役化: タップ=アンドゥ（1世代前に戻す）/ 長押し=全消去
- BPM同期ループ長: ドラムシーケンサーのBPMに合わせてループ秒数を自動計算（`loopSeconds` 変数化）

### 10. ビジュアライザ削除 + 3×2パッドグリッド
- `analyserNode` と描画ループ (`looperRAF`) のビジュアライザ部分を完全削除
- PENTA/MAJ モードの鍵盤を横3×縦2の大型パッドに変更（CSS Grid）
- パッドDOMorder `[3,4,5,0,1,2]` でグリッド上段=高音・下段=低音のランチパッド式配置

## トラブルシューティング
- **AudioWorklet + file:// protocol**: `addModule()` が AbortError → file://では動作しないがhttp経由（GitHub Pages）で正常動作。ScriptProcessorフォールバックが自動で機能
- **ステップボタン同化問題**: 黒背景に黒ボタンで見えない → TR-909風の明色キー+枠線+赤LEDドットに変更
- **パッド音程配置**: 上から下へ音程が下がるのが自然 → DOM順 `[3,4,5,0,1,2]` で解決
